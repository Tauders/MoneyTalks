Краткое описание:			
Управление счетами пользователем

Основное действующее лицо:		
Пользователь

Второстепенные действующие лица:	
Сервис учёта личных финансов

Предусловия:				
Открытие главной страницы Сервиса учёта личных финансов

Основной поток:
1. Вариант использования начинается, когда Пользователь открывает страницу управления счетами
2. Пользователь выбирает действие со счетом
3. Сервис отображает список измененных счетов

Альтернативные потоки:
	2.2. Пользователь выбирает опцию добавить счет
		2.2.1. Пользователь вводит название счета
		2.2.2. Сервис проверяет уникальность названия счета среди счетов Пользователя
		2.2.3. Пользователь указывает начальный баланс
		2.2.4. Сервис добавляет новый счет
	2.3. Пользователь выбирает опцию изменить счет
		2.3.1. Пользователь переименовывает счет
		2.3.2. Сервис проверяет уникальность названия счета среди счетов Пользователя
		2.3.3. Сервис сохраняет изменения
	2.4. Пользователь выбирает опцию удалить счет
		2.4.1. Сервис запрашивает подтверждение действия пользователя
		2.4.2. Пользователь подтверждает удаление счета
		2.4.3. Сервис удаляет счет
		
Постусловия:				
Нет
