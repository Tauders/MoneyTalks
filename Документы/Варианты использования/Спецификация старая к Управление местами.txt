Краткое описание:
Управление местами пользователя

Основное действующее лицо:
Пользователь

Второстепенные действующие лица:
Сервис учёта личных финансов

Предусловия:
Открытие главной страницы Сервиса учёта личных финансов

Основной поток:
1. Открытие страницы управления местами
2. Выбор действия с местом
3. Сервис отображает список измененных мест

Альтернативные потоки:
 2.1.   Пользователь выбирает опцию добавить место
  2.1.1. Пользователь вводит название места
  2.1.2. Сервис проверяет уникальность названия места среди мест пользователя
  2.1.3. Сервис добавляет новое место
 2.2.   Пользователь выбирает место, чтобы изменить его
  2.2.1. Пользователь переименовывает место
  2.2.2. Сервис проверяет уникальность названия места среди мест пользователя
  2.2.3. Сервис сохраняет изменения
 2.3.   Пользователь выбирает опцию удалить место
  2.3.1. Сервис запрашивает подтверждение действия пользователя
  2.3.2. Пользователь подтверждает удаление места
  2.3.3. Сервис удаляет место


Постусловия:
Нет.
