Описание классов
Класс "Пользователь" (User) предназначен для хранения личных данных пользователей системы.
Класс "Место" (Place) предназначенен для описания места совершения транзакций.
Класс "Категория" (Category) предназначен для хранения классификации транзаций.
Класс "Счёт" (Account) предназначен для хранения информации о средствах пользователя.
Класс "История" (History) предназначен для хранения истории транзакций, с указанием места, категории и суммы транзакции.
Описание взаимоотношений классов
Связь между "Пользователь" и "Счёт" необходима для привязки одного или нескольких счетов к конкретному пользователю системы.
Связь между "Пользователь" и "Место" необходима для создания каждому пользователю системы уникальных мест посещения.
Связь между "Пользователь" и "Категория" необходима для создания каждому пользователю уникального дерева категорий.
Связь между "Категория" и "Категория" необходима для создания иерархичной структуры категорий.
Связь между "Место" и "История" необходима для отображения места совершения транзакции.
Связь между "Категория" и "История" необходима для отображения категории совершённой транзакции.
Связь между "История" и "Счёт" "от" (to) необходима для отображения счёта, с которого была совершена транзакция.
Связь между "История" и "Счёт" "куда" (from) необходима для отображения счёта, на который была совершена транзакция.
